# 11.[北京]

偌大的办公室。宽敞的办公桌擦得一尘不染，桌后坐着一个人，他靠在椅背上，扫视着房间中的一切。椅子两旁，国旗和党旗低垂着。

一个人走进来，在桌前敬了个礼：“首长。”“查到了吗?” “还没有，对方似乎并不在大陆。”

“不可能。他不在大陆,还能在什么地方?我亲自下了命令，武警为什么没逮捕对方?”

“ ……他们解释说群众发生骚乱,他们一时找不到目标。”

“必须尽快找到对方，平息事态。”他的语气严肃起来,面部肌肉都随之紧绷了。“我通知了其他安理会常任理事国首脑，各国共产党和工人党大会开了紧急会议，我没有派人去。他们的态度不知道如何?”“联系一下 ‘金色黎明’？”桌前的人试探着问。

“尽量不要。上次希腊共产党就指责我们联系法西斯。”

“好吧，我再加大搜查力度，联系一下刑侦和公安部门。”

# 12.[西雅图]

“同志们，依照共产国际执行局的要求，党的机构要进行改组。这次召开的区部委员会就是要讨论这一事宜，并选举参加中央委员会的代表。”

一个代表放下手中的笔。“我们现在有什么方案吗?”约翰逊拿出一份文件说道：“目前的方案是建立支部、区部、总区部、中央委员会的党内机关，并受共产国际领导。此外,要利用这个机会清除党内的修正主义分子。”

另一个代表站起来，说道:“我不能接受这一方案!这样的改组将使我们失去大量成员,并招致外界的猜疑，而且成员的减少会让我们在市州等的机关中失去同情者，结果就是我们为了一个新的极权体制————所谓国际————陪葬!”

道恩也站起来。“什么极权体制?各国的工人运动要有统一的领导，这是历史的经验！所谓的同情者们，在革命真正爆发时能站在我们一边吗?我们要的是一个社会主义国家，而不是改良主义的施舍!”

"暴徒,各位!这是个恐怖分子!我们通过议会可以合法取得的成就,却被他说成非要起义不可。醒醒吧，战争的世纪已经过去了!我绝对不会和你们一起发这种疯！”

……

“约翰逊，来杯咖啡?

“谢谢。”

“现在怎么办呢?显然，有些人不接受改变。”

“放心，情况总会变好的。”

# 13.[深圳]

“同学们，”婷婷站起来，说道，“最近几个地方的工人要求建立工会，提高待遇,都被无理拒绝了。我上次还听我爸说，他们厂的工人要组建工会，他找市总工会的人说条件不够，就此不了了之。“建工会哪要什么条件?有工人还不够吗?毛主席倡导的工人管理一切的号召，如今成了老爷们的施舍了! ”一个学生不平地说。

“好了，开始咱们的话题吧。”“我来!” 一个学生站起来。“发生于半个世纪前的文化革命……”

门猛然被推开，一群穿着警服的人走进来。为首的一个说：“有人举报你们非法集会，传阅违法印刷品。请配合我们工作!”

几个警察走进屋子，翻看桌上的书。有学生想要上前阻止，却被一把拽住了。

“我们有什么罪?”婷婷气愤地说。“办读书会，学习马列毛著作，有错吗?”

为首的警察冷笑一声。“读书会?我看你们是受境外势力指使，蓄意挑动反对党和国家，图谋不轨!”“你有什么证据定我们的罪?你这是非法搜查、有罪推定!”

“非法？我们就是执法的！至于罪名，随便找一个，就把你们抓走！”

警察又看向婷婷。“小姑娘，挺厉害啊，跟我们走!”

# 14.[北京]

天安门广场上人流如织。不过，他们手中举着的并非相机，而是红旗和条幅，警察包围着广场，竭力把几于疯狂的记者挡在外面。

广场上不断有人大声质问：“毛主席现在在哪儿?”“你们把主席怎么了?”警察一言不发，只管站成一排，挡住内外两批人群。

红旗摇动，人们让出一小块区域，一位老人走来，高声喝问：“那天我亲眼看见了毛主席，你们怎么说没有这回事?当天就有警察来抓毛主席，还被我一顿骂！你说，你们是不是把毛主席关起来了？你说话啊！别以为不出声就能把我们糊弄过去!”

像是回应老人的问话，一辆黑色轿车在长安街中心停下，一个身穿西装的官员下了车。记者见状，立即拥向那人，却仍被警察挡住。

“毛泽东重生的消息是真的吗?”“政府为什么至今没有回应?”“纪念堂为什么突然关闭？”“本次事件与南方的多起罢工有没有关系?”没有人得到回答。

官员走到警察站成的人墙跟前,接过一个喇叭，说道：“市民们！请静一静！所谓的毛泽东同志重生是空穴来风，纪念堂关闭也属正常修缮，希望大家不要听信谣言，相信党和政府，尽快恢复正常的社会秩序！”

“不许打官腔!”“老实说！”官员的声音淹没在呼喊当中。

# 15. [乞拉朋齐]

六月的世界雨极，大雨从天幕中倾泻而下，但仍有不少人聚集在广场上，任由雨水打湿身上的迷彩服。

广场上有一个木板搭起的平台，悬挂着毛主席像。其上是一张红色条幅，写着“印度共产党（毛主义）大会”。

主席坐在台上，一个人跑上台，和他交谈了几句。主席随即向台下宣告：“同志们，我们的队伍已经击溃了最后一股敌军。他们现在正向这里行进！”台下一片欢呼声。

“同志们，”主席挥了挥手。“在毛泽东思想的指导下，我们成功地策划了因帕尔、乞拉朋齐、丁苏吉亚的起义，并与农村斗争结合，最终夺取了东部地区的绝对控制权。虽然这块区域只占印度的一小部分，但这是我们革命的伟大的一步！

“我们的党在马列主义，毛泽东思想的旗帜下，从红色走廊的斗争到如今，占领了包括城市在内的广大区域。而在将来，我们还必将扩大我们的胜利，并和全世界工人农民一道，反对帝国主义、修正主义，让赤旗插遍全球！”雨这时渐渐小了，台下的人们高兴地整理自己被淋湿的衣物，一边聆听着台上胜利的讲演。

“现在，我宣布，印度革命政府成立了!”

雨霁云散，金色的阳光映照在四方招展的红旗上。

# 16.[北京]

一个人靠在沙发靠背上，凝视着国旗和党旗之间空荡荡的办公椅。办公室锁了门，他也关照过不让人进来，因此他可以放松地靠在沙发上。其实这是他思考时最舒服的姿势。

恍惚间，他忘了自己处在的城市，周围似乎是半个世纪前他熟悉又陌生的桌椅，墙上的标语、画像仿佛重现在眼前。这是他家，当时还在大院里。

“走，去兜一圈?”窗外是同一个大院的好友。“走！”两人跨上闪亮的自行车，穿梭在胡同街巷里。“一会回去，去我家看电视，我家还有汽水呢！”

场景变换。客厅的柜子上，多了一尊白瓷半身像。他走进门，一身军绿，右臂的袖章上“红卫兵”三个字十分显眼。

父母都不在家，不过老方、平平他们都在。他把帽子挂在衣架上，招呼着客人，虽然他们已经很熟了。“今天怎么样?”老方问。“还行,他们又说要学毛选，我也不让，说他们不配呢，结果让他们在教室墙角蹲了一天！”

平平兴奋地接过话头，“今天我在街上看见一伙人，一看就是工人家的，骑着自行车一边背语录，一边喊什么，啊，对了‘主席对不对，英不英明，咱们十年后见’！”

老方说道：“好，他们越这样，对咱们就越有利。反正也动不到咱们头上来！”三人都笑了。

眼前一闪，他又回到了现实中自己的办公室。他站起来坐到桌前，拿起一张纸写起来。“对于目前印度……我国西南……可能的威胁……全军进入紧急状态，在各地……加强戒备,尤其……国内不法分子……制造混乱！”

# 17.[香港]

“情况不太好，主席。”蒯大富忧心忡忡地说。“我听说北京那边下了命令，说是印度不安稳，要各地加强警戒，军队加强训练。说是冲着印共毛，我看更像冲着您。”

“大陆有什么动静啊?”“您的事已经传开了，可官方死活不承认。为这事，天安门广场,延安,湘潭都有人游行，有的还跟警察对峙起来了。南方罢工的事情也不算少，听说深圳还抓了几个办读书会的学生呢。”

“有这样的事?”毛主急切起来。“那为什么全国人民不联合起来?”“消息都被封锁了，除了看见的，没什么人知道，在香港消息还灵通一点。再说了，谁也不愿意出头，我们都老了。”

“那学生干了什么？” “唉，”蒯大富苦笑一声，“跟我们样，反帝反修，文化革命。您和我都是亲历者，可苦了孩子们了。”

毛主席握紧了拳头。“真是乱搞！好好的一个国家搞成了法西斯！照这个样子，他们也离灭亡不远了！”

蒯大富说：“ 时间一长，香港也有危险。主席您或许可以去印共毛那边避避。”

深夜的渡口一片寂静，只有微弱的灯光的光照在海面上。蒯大富目送毛主席上船，在码头上，立正敬礼。“主席保重！有朝一日，我和您还会再见！”

# 18. [深圳]

婷婷穿行在粉刷得雪白的走廊里，身后跟着一个警察。虽然没有人扳着，但她还是感到了巨大的压抑和不平。“到了，这间。”警察打开门，婷婷也跟了进去坐下。

椅背很硬，婷婷一坐下就感到全身酸痛。被拘留的这几天里，她频繁地被人叫走看那些乏味的记录片，所谓“教育改造”。还不断有人故意弄出声音，打扰她时间本就不长的睡眠。巨大的身心压力让她健康失调，反复要求下，对方竟然只给了自己一个空药瓶！现在的她，早已疲惫不堪，只是竭力支持着思维。

“我到底犯了些什么法?能告诉我吗?”对面的警察不回答。拿出一张纸递到地面前。“签字！” 他又把那张纸往婷婷面前推了推。

婷婷拿起来。“非法经营？真是笑话！你们原来不是说‘非法集会’吗？我经营了些什么？我不能签字！”

“口供、物证俱在，你不签也得签！”警察把一堆东西放上桌面。

“你们……伪造口供！”婷婷气愤地大喊。“我没说过这些！还有，你们什么时候去了我家？搜查证呢？”

“放老实点！”警察也板起脸来。“你现在是嫌犯身份，坦白从宽，抗拒从严！”

“要我坦白什么？你们没有罪名就抓了我，伪造了我的口供，违规搜查我家，还要我坦白什么？”

“你不说，我迟早会让你说！走，回去！”警察又把婷婷带出了房间。

# 19. [乞拉朋齐]

一辆越野车驶入城区，在一栋楼前停下。车身散布着泥点与树叶，显然刚刚穿越了山地丛林。门口的卫兵转身进去报告，一会儿，一个身穿迷彩服的干部与卫兵一同走了出来。

干部打开车门，毛主席从车上下来了。“你们好啊！”毛主席向干部和卫兵招了招手。干部和卫兵忍住激动的心情，立正敬礼。“主席好！”他们把毛主席带进楼里。

“主席，我们的起义成功不久，又经过反动派的破坏，城市还破败得很。您的住处就安排在这附近，也更安全些。虽然我们尽力保密了，但难保时间长了，敌人会知道您的行踪。”

“我住惯了山沟窑洞了，条件艰苦一点没有什么。你们目前的斗争情况怎么样了？”

“起义胜利后，我们尽快地废除旧政府部门，建立行政委员会，加强军队建设。而且在农村，我们号召农民群众揭发种姓制度、分配地主土地，取得了一些成效。”

“好啊，印度人民站起来了！”毛主席大手一挥，“希望你们继续斗争，继续革命，打倒全世界反动派！”

# 20.[深圳]

这是一间空荡荡的教室。虽然读书会已经被强制停办，但老师仍然允许他们借用这间教室。

“婷婷有消息了吗？”一个学生回答：“没有。我和其他几个同学去问过不少次了，可他们就说还没有办结，罪名也不肯说。”

“真是荒谬！可惜婷婷一个女孩子，还不知要受多少迫害呢！”

“对了，”那个学生打开笔记本电脑，调出一个窗口。“我在网上结识了这个约翰逊，他自称是美共党员。你来看一看。”

对方此时恰好在线，于是两个学生发送了条消息：“您好!我们是中国的学生。您能否向我们介绍一些国际上的情况？谢谢！”

不久就有了回复：“当然可以！你们知不知道毛主席重生的事？我们已经建立了新的共产国际，并改组了党。哦，还有，印共毛在东部建立了革命政权，我也是最近才知道的。”

两个学生交换了惊讶的神色，原来无产阶级革命已经悄然发生了！“我们的同伴由于学习马列主义遭到了非法逮捕，您有什么建议吗？”

“我听说了最近中国的一些罢工和群众运动。我认为你们应该先寻求可靠的同志，和工人、农民中的先进分子建立稳固的联系。”

“对！”两个学生异口同声地说。
